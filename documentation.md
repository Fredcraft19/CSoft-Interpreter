# C~
## Semi-Colons
For semi-colon usage, use it identically to C#.
## Variables
### Declaration
To declare a variable in C~ just do:
```cs
var x = 5;
```
You do not need to specify its type. The variables are like in Javascript where you can have `x = "hi"` then `x = 5`

### Change value
To change the value of a varaible just do:
```cs
x = 9;
```

## IF & WHILE statements
To create an IF/WHILE statement, just do:
```cs
if(x > 5){
    var output = "x > 5";
}
```
or
```cs
while(i > 1){
    var output = "This is a while loop.";
}
```
You can write the IF statements like in any other language hopefully

## METHODS
To declare a method, just do:
```cs
method methodName(parameter1, parameter2){
    // code here
}
```
To call a method, do:
```cs
methodName(2, 3);
```
